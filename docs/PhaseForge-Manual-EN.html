1. Overview

PhaseForge is a desktop application for loudspeaker measurement analysis, target curve design, parametric EQ optimization, and FIR filter generation. It supports multi-band crossover systems with linked HP/LP filters, interactive PEQ editing, and export of correction filters as WAV impulse responses.

Key Features

  • Import REW (.txt) and FRD measurements with magnitude and phase
  • Near-field + far-field measurement merge with baffle step correction
  • Configurable target curves: HP/LP filters (Butterworth, Bessel, LR, Gaussian), tilt, shelving
  • Automatic PEQ optimization (greedy peak-matching + LMA refinement)
  • Interactive PEQ editing: drag bands, scroll for Q, double-click to create
  • Multi-band crossover system with linked filters
  • FIR filter generation with iterative WLS optimization
  • Configurable narrowband boost limiting and frequency weighting
  • Export FIR as WAV (mono, float64)
  • Project save/load with full state preservation

2. Getting Started

  1. Launch PhaseForge. The Welcome dialog offers New Project or Open Project.
  2. Create a new project: enter name, choose band count (1–8).
  3. On the Measurement tab, click Import to load a .txt or .frd file.
  4. Switch to Target tab to configure target response (HP, LP, tilt, shelves).
  5. Click Optimize in the PEQ sidebar to auto-generate PEQ correction.
  6. Switch to Export tab to preview and export the FIR filter.

3. Interface Layout

The interface is divided into:

  • Top toolbar — file menu, project name, strategy toggle, Settings, Optimize All
  • Band tabs — SUM + individual band tabs (draggable to reorder)
  • Plot area — upper: frequency response; lower: impulse / PEQ / export impulse
  • Control panel (right) — measurement, target, or export controls
  • PEQ sidebar (right, Target tab only) — PEQ band table and optimizer
  • Status bar (bottom) — band count, current view

5. Band Tabs & SUM View

Band Management

  • SUM tab (always first) — shows combined response of all bands
  • Band N tabs — individual band workspaces
  • + button — add new band
  • × on tab — delete band (with confirmation)
  • Drag tabs to reorder bands

SUM View Features

  • All band target curves and corrected responses overlaid
  • Crossover markers at band boundaries
  • Drag crossover markers to adjust frequency
  • Double-click crossover to open detailed Crossover Dialog

6. Measurement Tab

  • Import File — load .txt (REW) or .frd measurement
  • Merge NF+FF — blend near-field and far-field measurements
  • Smoothing — Off, 1/3, 1/6, 1/12, 1/24 octave, or Variable
  • Delay compensation — auto-detects propagation delay from phase data; toggle to remove
  • Polarity invert — NOR/INV toggle (+180° phase shift)
  • Floor bounce — reflection simulation (speaker height, mic height, distance)

7. Target Tab

General

  • ON/OFF toggle for target curve
  • Level (dB) — reference level adjustment
  • Tilt (dB/octave) — frequency slope for house curve

Band Linking

Click the link button (&link;) on LP to link with the next band's HP. When linked, changing one automatically mirrors the other (same type, order, frequency).

Low/High Shelf

Optional shelf filters with frequency, gain, and Q parameters.

11. Dialogs

Optimization Settings

Opened via the Settings button in the toolbar.

ParameterRangeDescription
WLS iterations0–20Iterative error correction passes. 0 = off, 3–5 optimal
Frequency weightingon/offHigher priority for crossover and speech bands (200–4k Hz)
Narrowband limitingon/offClamp sharp correction peaks above smoothed curve
NB smoothing0.05–2 octSmoothing window. Smaller = more precise
NB max excess1–24 dBMax allowed boost above smoothed correction
Max boost0–60 dBGlobal correction boost limit
Noise floor−200 to −40 dBIgnore correction below this level. Raise for noisy measurements

Crossover Dialog

Double-click a crossover marker in SUM view. Configure filter type, order, frequency, and linear phase for the LP/HP pair.

Merge NF+FF Dialog

Blend near-field and far-field measurements:

  • Select NF and FF files
  • Splice frequency (50–1000 Hz)
  • Blend width (0.5–3.0 octaves)
  • Auto level offset (or manual override)
  • Optional baffle step correction

Baffle Step Dialog

Configure baffle dimensions and driver position. Live preview shows correction curve with f3 frequency and edge diffraction points.

12. Project Management

Project Structure

Each project is stored as a folder containing:

  • project-name.pfproj — JSON metadata file
  • inbox/ — imported measurement files
  • All band configurations, PEQ bands, target curves, and optimization settings
