1. Обзор

PhaseForge — десктопное приложение для анализа измерений акустических систем, проектирования целевых кривых, автоматической оптимизации параметрического эквалайзера и генерации FIR-фильтров коррекции. Поддерживает многополосные кроссоверные системы со связанными фильтрами HP/LP, интерактивное редактирование PEQ и экспорт импульсных характеристик в формате WAV.

Ключевые возможности

  • Импорт измерений REW (.txt) и FRD с амплитудой и фазой
  • Сшивка ближнего и дальнего поля (NF+FF) с коррекцией баффл-степа
  • Настраиваемые целевые кривые: HP/LP фильтры (Butterworth, Bessel, LR, Gaussian), наклон, полки
  • Автоматическая оптимизация PEQ (жадный поиск пиков + LMA-уточнение)
  • Интерактивное редактирование PEQ: перетаскивание полос, прокрутка для Q, двойной клик для создания
  • Многополосная кроссоверная система со связанными фильтрами
  • Генерация FIR с итеративной WLS-оптимизацией
  • Настраиваемое ограничение узкополосного буста и частотное взвешивание
  • Экспорт FIR в WAV (моно, float64)
  • Сохранение/загрузка проектов с полным сохранением состояния

2. Быстрый старт

  1. Запустите PhaseForge. Приветственное окно предлагает New Project или Open Project.
  2. Создайте проект: введите имя, выберите количество полос (1–8).
  3. На вкладке Measurement нажмите Import для загрузки файла .txt или .frd.
  4. Перейдите на вкладку Target для настройки целевой кривой (HP, LP, наклон, полки).
  5. Нажмите Optimize в боковой панели PEQ для автоматической генерации коррекции.
  6. Перейдите на вкладку Export для просмотра и экспорта FIR-фильтра.

3. Компоновка интерфейса

  • Верхняя панель — меню файла, имя проекта, переключатель стратегии, Settings, Optimize All
  • Вкладки полос — SUM + вкладки отдельных полос (перетаскиваемые)
  • Область графиков — верхний: АЧХ/ФЧХ; нижний: импульс / PEQ / экспорт
  • Панель управления (справа) — настройки измерения, цели или экспорта
  • Панель PEQ (справа, только Target) — таблица PEQ-полос и оптимизатор
  • Строка состояния (внизу) — количество полос, текущий режим

5. Вкладки полос и режим SUM

Управление полосами

  • SUM (всегда первая) — суммарная характеристика всех полос
  • Band N — рабочие области отдельных полос
  • + — добавить новую полосу
  • × на вкладке — удалить полосу (с подтверждением)
  • Перетаскивание вкладок для изменения порядка

Режим SUM

  • Наложение целевых и скорректированных кривых всех полос
  • Маркеры кроссоверов на границах полос
  • Перетаскивание маркеров для настройки частоты кроссовера
  • Двойной клик по маркеру — открытие диалога кроссовера

6. Вкладка «Measurement»

  • Import File — загрузка измерения .txt (REW) или .frd
  • Merge NF+FF — сшивка ближнего и дальнего поля
  • Smoothing — Off, 1/3, 1/6, 1/12, 1/24 октавы или Variable
  • Компенсация задержки — автоопределение задержки по фазе; переключатель для удаления
  • Инверсия полярности — NOR/INV (+180° сдвиг фазы)
  • Floor bounce — моделирование отражений (высота динамика, микрофона, расстояние)

7. Вкладка «Target»

Общие параметры

  • ON/OFF — включение/выключение целевой кривой
  • Level (дБ) — опорный уровень
  • Tilt (дБ/октава) — наклон (house curve)

Фильтры HP / LP

ПараметрЗначения
Тип фильтраButterworth, Bessel, Linkwitz-Riley, Gaussian
Порядок1–8 (LR: только 2, 4, 8)
Частота10–20 000 Гц
Linear PhaseЧекбокс (всегда вкл. для Gaussian)
M (только Gaussian)Коэффициент формы, 0.5–10

Связка полос

Нажмите кнопку связи (&link;) на LP, чтобы связать с HP следующей полосы. При связке изменение одного автоматически зеркалит другой (тип, порядок, частота).

Полочные фильтры (Shelf)

Опциональные low/high shelf с параметрами частоты, усиления и Q.

8. Вкладка «Export»

Конфигурация FIR

ПараметрВарианты
Sample Rate44.1k, 48k, 88.2k, 96k, 176.4k, 192k Гц
Taps4K, 8K, 16K, 32K, 64K, 128K, 256K
Window20+ типов: Hann, Blackman-Harris, Kaiser, Dolph-Chebyshev и др.

Отображение

  • Верхний график: модельная АЧХ/ФЧХ vs. реализованная FIR АЧХ/ФЧХ
  • Нижний график: импульсная характеристика FIR (временная область)
  • Строка статуса: количество тапов, частота дискретизации, окно, режим фазы, нормализация, каузальность %
  • SNAP/CLR: снимок текущей кривой для сравнения
  • Export WAV: сохранение импульсной характеристики FIR в файл

Кнопки управления графиками

  • SNAP — снимок текущей кривой для наложения
  • CLR — очистить все снимки
  • FIT — автомасштабирование по данным
  • +/− — зум по каждой оси
  • ▵/▿/◃/▹ — панорамирование

Разделитель графиков

Перетаскивайте горизонтальную полосу между верхним и нижним графиками для изменения пропорций (15%–65%).

10. Боковая панель PEQ

Отображается на вкладке Target (не в режиме SUM).

Авто-подбор

  • Tolerance (дБ) — допуск ошибки оптимизации (0.5–3.0)
  • Max bands — лимит количества PEQ-полос (1–60)
  • Optimize — запуск оптимизатора для текущей полосы
  • Clear — удалить все PEQ-полосы

Таблица полос

СтолбецОписание
ЧекбоксВключение/выключение полосы
Freq (Гц)Центральная частота, 20–20 000. Колесо мыши для настройки
Gain (дБ)±18 дБ (Standard) или ±60 дБ (Hybrid). Цветовая индикация
QДобротность, 0.1–20. Колесо мыши для настройки
×Удалить полосу

11. Диалоговые окна

Настройки оптимизации (Optimization Settings)

Открывается кнопкой Settings на панели инструментов.

ПараметрДиапазонОписание
WLS iterations0–20Итеративная коррекция ошибок. 0 = выкл, 3–5 оптимально
Frequency weightingвкл/выклПовышенный приоритет кроссоверных и речевых полос (200–4k Гц)
Narrowband limitingвкл/выклОграничение острых пиков коррекции выше сглаженной кривой
NB smoothing0.05–2 октОкно сглаживания. Меньше = точнее, больше = агрессивнее
NB max excess1–24 дБМакс. допустимый буст над сглаженной коррекцией
Max boost0–60 дБГлобальный лимит усиления коррекции
Noise floor−200..−40 дБКоррекция ниже этого уровня игнорируется. Поднять для шумных замеров

Диалог кроссовера (Crossover)

Двойной клик по маркеру кроссовера в SUM. Настройка типа фильтра, порядка, частоты и линейной фазы для пары LP/HP.

Сшивка NF+FF (Merge)

  • Выбор файлов NF и FF
  • Частота сплайса (50–1000 Гц)
  • Ширина бленда (0.5–3.0 октавы)
  • Автоматический уровень (или ручной)
  • Опциональная коррекция баффл-степа

Баффл-степ (Baffle Step)

Настройка размеров панели и позиции динамика. Предпросмотр кривой коррекции с частотой f3 и точками дифракции.

12. Управление проектами

Структура проекта

  • project-name.pfproj — JSON-файл метаданных
  • inbox/ — импортированные файлы измерений
  • Все конфигурации полос, PEQ, целевые кривые и настройки оптимизации

Рабочий процесс

ДействиеСочетание
Новый проектCmd+N
Открыть проектCmd+O
СохранитьCmd+S
Сохранить какShift+Cmd+S
Примечание: FIR-фильтры не сохраняются в проекте — они пересчитываются по запросу. Используйте Export WAV на вкладке Export для сохранения.
